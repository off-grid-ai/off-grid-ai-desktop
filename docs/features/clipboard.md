# Clipboard (Pro)

[← All features](../FEATURES.md)

A private clipboard manager. Everything you copy is kept in a searchable history with
previews, and a global-hotkey popup pastes any past item back in two keystrokes. It all
lives **on your device** - nothing is synced or uploaded.

- **Searchable history** of everything you copy - fuzzy search across past clips.
- **Image, PDF, and file previews** - not just text; copied files keep their bytes so
  you can paste the real file again.
- **Quick-paste popup** on a global hotkey (**Cmd+Shift+C**) - pick a clip and it pastes
  into the app you were in.
- **Retention controls** - cap the history size and auto-expire old clips.
- **On-device** - the history is stored locally; capture can be paused any time.

→ Part of [Off Grid Pro](pro.md).
